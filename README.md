# pygmy
Take a list of urls and fatch for some types of api keys in the source code of each one using regex.

Usage:

```
▶ git clone https://github.com/g0ttfr1d/pygmy

▶ cd pygmy

▶ python3 pygmy.py -f <urls_list.txt>
```
